# UniCore Mobile App

React Native (Expo) mobile application for students and staff.

## Setup

```bash
cd mobile
npm install
expo start
```

## Features

- Student dashboard
- Attendance QR scanner
- Course materials
- Results
- Notifications

## Structure

```
src/
├── screens/       # Screen components
├── components/    # Reusable UI
├── services/       # API services
├── store/         # State management
└── utils/         # Helpers
```
